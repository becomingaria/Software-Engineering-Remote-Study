# Learn Canvas

An introduction to the HTML5 Canvas API through guided exercises.

## Getting Started

Open `part-one/index.html` in your browser using Live Server (VS Code extension) or by double-clicking the file.

## Structure

```
learn-canvas/
├── part-one/     ← Introduction: shapes and colors
├── part-two/     ← Drawing a smiley face
└── part-three/   ← Cube Crusher interactive challenge
```

## Prerequisites

- HTML & CSS basics
- JavaScript DOM manipulation

## How to Use

Work through each part in order. Each folder has an `index.html`, a `styles.css`, and a `script.js` (or `challenge.js` for part-three). Follow the instructions in the README for the relevant section or in the course notes.
