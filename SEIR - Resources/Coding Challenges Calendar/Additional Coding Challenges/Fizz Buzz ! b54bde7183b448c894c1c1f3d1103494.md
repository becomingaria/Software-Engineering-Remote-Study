# Fizz Buzz !

Given an integer `n`, write a function that returns an *an array of strings* `answer` *where*:

Each element in the `answer` array should either be a ‘fizzbuzz’, ‘fizz’, ‘buzz’, or the current number (as a string). 

- `"FizzBuzz"` when a number is divisible by `3` and `5`.
- `"Fizz"` when a number is divisible by `3`.
- `"Buzz"` when a is divisible by `5`.
- `i` - the current number **(as a string)** if none of the above conditions are true.
