# Graphs

## Overview

In this lesson, we cover the basics of the node-based data structures known as graphs.

There is one component to this lesson:

1. In-Class Exercise: Implementing a Graph
    >

## Learning Objectives

By the end of this lesson, you'll be able to:

- Define a graph and the difference between undirected and directed graphs.
- Identify how graphs are used in programming.
- Implement a graph.

- Linked lists
- Binary trees

## Implementing a Graph

Implement a graph and the methods outlined in the `Graph.js` file.



```jsx
class Graph {
    constructor() {
        // Create a property called `nodes` and set it equal to an empty object.
        // This will be our adjacency list.
    }

    addNode(node) {
        // If the node value passed in does not already exist in our adjacency
        // list, then add it as a key and set it equal to an empty array.
    }

    addEdge(node, edgeNode) {
        // If the node exists in our adjacency list, then push the edge into the
        // array of edges for that node.
    }
}
```
