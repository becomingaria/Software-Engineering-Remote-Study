# Birthday Cake Candles

**Prompt**: You are in charge of the cake for a child's birthday. You have decided the cake will have one candle for each year of their total age. The height of the candles are varied, and the height is indicated by integers. Find the height of the tallest candle, and how many candles on the cake are that height.

For Example, if `candles = [4,4,1,3]` the maximum height candles are **4** units high. There are **2** of them, so return **2**.

Example Input: 

```jsx
const candles = [4,4,1,3]
```

Example Output:

```jsx
2
```

# **Function Description**

**Function should be called `birthdayCakeCandles` and have the following parameter(s):**

- *candles[n]*: the candle heights

### Input Format

Input will be an array of integers signifying the height of candles. 

### Output Format

Return the number of candles that are the tallest as an integer. 

**Returns**

- *int*: the number of candles that are tallest
