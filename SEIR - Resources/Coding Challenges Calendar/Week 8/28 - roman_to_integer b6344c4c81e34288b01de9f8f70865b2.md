## Solution

```python
def roman_to_int(s):
        map_romans = {
            "I": 1,
            "V": 5, 
            "X": 10,
            "L": 50,
            "C": 100,
            "D": 500, 
            "M": 1000
        }
        num = 0
        value = 0
        for letter in s[::-1]:
            preValue = value
            value = map_romans.get(letter)
            if value <= preValue:
                if (preValue == value * 5 or preValue == value * 10):
                    num -= value
                else:
                    num += value
            else:
                num += value
        return num
```
