# Django Tutorial Lab

For the remainder of today’s activities you will work solo, or in small groups to independently build a Django web application. The lab will follow a step-by-step tutorial that introduces you to the basic syntax, principles, and design approaches found in the Django web framework. 

You can begin the tutorial by following the steps outlined in the Django project’s documentation. 

[Django](https://docs.djangoproject.com/en/4.1/intro/tutorial01/)

The tutorial will cover the following phases of development:

Part 1 - Creating a project and first view

Part 2 - Adding models + database

Part 3 - More views + Django Template system

Part 4 - Forms + class based Views

Part 5 - (skip for now) Testing your django app 

Part 6 - Styling the app and static files

The purpose of this lab is to expose you to the basics of Django before we dive deeper into the framework over the next series of demonstrations.
