# Intro to Data Structures

# Stacks

## Overview

Peanut butter and jelly, Batman and Robin, Rick and Morty... stacks and queues are another iconic duo. For this mornings exercise we will focus on stacks.

There is one component to this lesson:

1. [In-Class Exercise: Bracket Matching](Intro%20to%20Data%20Structures/Bracket%20Matching%20Exercise%203d307ef70a4746f99c4f77f608937eb8.md)

## Learning Objectives

By the end of this lesson, you'll be able to:

- Distinguish between a stack and a queue.
- Determine situations in which you’d use a stack or queue over another data structure.
- Build a stack and a queue using a linked list or an array.

## Prerequisites

- Big O notation
