## Welcome! 👋

**To do this challenge, you need a basic understanding of HTML and CSS.**

## The challenge

Your users should be able to:

- See a different layout depending on the size of the screen.
- See a hover state on desktop for the Sign Up call-to-action

## Where to find everything

Your task is to build out the project to the designs shown below. You will find mockups of both a mobile and a desktop version of the design below. 

### Mobile First Approach

Start by building the mobile version of the layout first using a new Replit or JS Code Sandbox.

After the mobile is looking good, and if you still have time, add media queries to your CSS to modify the layout of your component to match the desktop view.

The designs are in JPG static format. Using JPGs will mean that you'll need to use your best judgment for styles such as `font-size`, `padding` and `margin`.

Have fun building! 🚀

## Front-end Style Guide

## Layout

The designs were created to the following widths:

- Mobile: 375px
- Desktop: 1440px

## Colors

### Primary

- Cyan: hsl(179, 62%, 43%)
- Bright Yellow: hsl(71, 73%, 54%)

### Neutral

- Light Gray: hsl(204, 43%, 93%)
- Grayish Blue: hsl(218, 22%, 67%)

## Typography

### Body Copy

- Font size: 16px

### Font

- Family: [Karla](https://fonts.google.com/specimen/Karla)
- Weights: 400, 700
